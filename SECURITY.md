# Security Policy

## Supported versions

Security fixes are provided for:

- latest `main`
- latest tagged release

Older unsupported snapshots may be left unpatched.

## Reporting a vulnerability

Please report vulnerabilities privately:

- https://github.com/PetriLahdelma/toolbox-starter-node-cli/security/advisories/new

Please include:

- affected version or commit
- proof of concept or clear reproduction steps
- impact analysis and severity estimate
- suggested mitigation (if available)

Avoid publishing exploit details in public issues or pull requests.
