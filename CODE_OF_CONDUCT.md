# Code of Conduct

## Our commitment

This project is committed to a respectful, professional collaboration environment.

## Expected behavior

- Be constructive and specific.
- Focus feedback on code and behavior, not people.
- Keep discussions outcome-focused.

## Unacceptable behavior

- Harassment, discrimination, threats, or personal attacks.
- Publishing private or sensitive information without consent.

## Reporting

Please report issues privately to the maintainer listed in repository metadata.
